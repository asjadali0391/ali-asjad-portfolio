# Muhammad Ali Asjad Portfolio

Static GitHub Pages portfolio for data analyst and data scientist roles.

## Files

- `index.html` - main portfolio page
- `styles.css` - responsive visual design
- `script.js` - small enhancement for icons and footer year
- `assets/Muhammad-Ali-Asjad-Resume.pdf` - downloadable resume

## Publish on GitHub Pages

1. Create a new public GitHub repository named `ali-asjad-portfolio`.
2. Push this folder to the repository.
3. In GitHub, open `Settings > Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select `main` and `/root`, then save.

Useful push commands after the empty GitHub repo exists:

```bash
git remote add origin https://github.com/asjadali0391/ali-asjad-portfolio.git
git push -u origin main
```

Your site will publish at:

`https://asjadali0391.github.io/ali-asjad-portfolio/`
